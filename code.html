Code

Discriminative Probabilistic Prototype Learning: Code implementing the DPPL method of Bonilla and Robles-Kelly (ICML 2012). See the paper for details.

Multi-task Gaussian Processes: Code implementing the MTGP method of Bonilla, Chai and Williams (NIPS 2008). See the paper for details.

Regularized Topic Models: Code implementing the RTM method of Newman, Bonilla and Buntine (NIPS 2011). See the paper for details. To run the example you will need the Blogs climate dataset.

Gaussian Process Preference Elicitation: Code implementing the GPPE method of Bonilla, Guo and Sanner (NIPS, 2010). See the paper for details.

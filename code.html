Code

There is publicly available code for most of my machine learning papers on my publications web page. This is the list of software I mantain:

Extended and Unscented Kitchen Sinks: Code implementing the Extended and Unscented Kitchen Sinks method of Bonilla et al (ICML 2016). See the paper for details.

Discriminative Probabilistic Prototype Learning: Code implementing the DPPL method of Bonilla and Robles-Kelly (ICML 2012). See the paper for details.

Multi-task Gaussian Processes: Code implementing the MTGP method of Bonilla, Chai and Williams (NIPS 2008). See the paper for details.

Regularized Topic Models: Code implementing the RTM method of Newman, Bonilla and Buntine (NIPS 2011). See the paper for details.

Gaussian Process Preference Elicitation: Code implementing the GPPE method of Bonilla, Guo and Sanner (NIPS, 2010). See the paper for details.
